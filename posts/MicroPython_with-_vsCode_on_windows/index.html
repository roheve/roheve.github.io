Posted Jul 26, 2022 Updated Sep 11, 2022
By Roelof Heuvel
1 min read

Using micropython with visual studio code

Found an article series for configuring and using vscode for micropython and connecting it to raspberry pico.

The first article is about configuring your windows laptop with the tools you need. The Getting Started post contains the installation steps you need to follow.

The second article explains how to use this for coding. This is the Start Coding post and explains it with a ‘Hello World’ example.

A third article has hints on how to configure git to use vscode for it’s editor. I am not so sure if I need this, as you can use git from within vscode, but why not configure it both ways.
